# Meditation Guide AI MCP

Deliver guided meditation scripts, breathing exercises, and mindfulness timers.

## Tools

- `get_breathing_exercise` — Return a structured breathing exercise.
- `get_meditation_script` — Return a short meditation script by theme.
- `suggest_mindfulness_reminder` — Return a random mindfulness reminder.

## Install

```bash
pip install -e .
```

## Run

```bash
python -m meditation-guide-ai-mcp.server
```
